# network-book-reader
通过爬取新笔趣阁的内容， 实现小说阅读app 的 一个安卓项目

## 项目背景
基于现在市面上小说软件大多有很多广告，所以我想自己开发一个 无广告、适合自己使用的小说阅读软件。

小说来源是 通过爬取新笔趣阁来获取。
